# Dealer Authentication Portal

A comprehensive dealer management system built with Next.js, featuring secure authentication, product management, and category management with dealer-based authorization.

## Features

- **User Authentication**: Secure login with email and password
- **Session Management**: Persistent login sessions with localStorage
- **Protected Routes**: Automatic redirection for unauthenticated users
- **Dealer Profile**: Display dealer name and email in dashboard
- **Logout Functionality**: Secure logout with session cleanup
- **Dashboard**: Complete product and category management
- **CRUD Operations**: Full create, read, update, delete for products and categories
- **Dealer Authorization**: Only dealers can modify their own products/categories
- **Modern UI**: Responsive design with Tailwind CSS
